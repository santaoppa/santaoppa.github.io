# 권한 모드

### `default`
- **표준 동작 모드**
- 각 도구를 처음 사용할 때마다 사용자에게 권한을 요청

### `acceptEdits`
- **자동 편집 승인 모드**
- 현재 세션에서 파일 편집 권한을 자동으로 승인

### `plan`
- **계획(Plan) 모드**
- 프로젝트를 분석하고 계획을 제안
- 파일 수정이나 명령 실행은 하지 않음

### `bypassPermissions`
- **권한 확인 생략 모드**
- 모든 권한 요청을 건너뛰고 작업 수행
- 안전한 개발 환경에서만 사용하는 것을 권장


<br>
<br>

# Plan 모드 사용 예시
- 코드 리뷰
- 아키텍처 설계
- 버그 분석

<br>
<br>

# 실전 워크플로우
1. Plan 모드로 계획 세우기
2. 계획 검토
- 마음에 안드는 부분이 있으면 계속 수정 요청
3. 만족할 때까지 반복
4. 최종 계획 확정 후 구현
- **Accept Edit**모드로 자동 구현 (모드 전환  `Shift+Tab`)

<br>

> 💡 **구현 후 수정 요청 방법**  
> 스크린샷으로 피드백 주기
